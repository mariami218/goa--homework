რას ნიშნავს რესპონსივი?

რესპონსიული ვებდიზაინი (Responsive Web Design) არის ტექნიკა, რომელიც საშუალებას აძლევს ვებსაიტს ავტომატურად მოერგოს სხვადასხვა ზომის ეკრანებს. ეს უზრუნველყოფს მომხმარებლის უკეთეს გამოცდილებას, რადგან საიტის შინაარსი ადაპტირდება და ოპტიმიზირდება როგორც კომპიუტერებისთვის, ასევე პლანშეტებისა და მობილურებისთვის.

რესპონსიული დიზაინის მთავარი უპირატესობა ის არის, რომ საჭიროა მხოლოდ ერთი ვერსია საიტისა, რომელიც ყველა მოწყობილობაზე სწორად გამოჩნდება. მასში გამოიყენება CSS მედია მოთხოვნები (@media queries), მოქნილი განლაგება (flexbox, grid) და მოქნილი გამოსახულებები.
