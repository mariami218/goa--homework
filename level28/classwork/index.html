1) განსხვავება inline, internal და external CSS ტექნიკებს შორის: Inline CSS: სტილები დაწერილია უშუალოდ HTML ელემენტში style ატრიბუტის გამოყენებით. გამოიყენება მხოლოდ კონკრეტულ ელემენტზე. მაგალითი:

ეს ტექსტი წითელია.

დადებითი: სწრაფი და მარტივი ერთჯერადი ცვლილებებისთვის. ნაკლოვანება: რთული ხდება დიდი პროექტების მართვა, რადგან სტილები HTML კოდშია და არა ცალკე ფაილში. Internal CSS: სტილები დაწერილია HTML დოკუმენტის დადებითი: ყველა ელემენტისთვის ერთ გვერდზე უფრო მარტივია სტილების მართვა. ნაკლოვანება: არ არის ხელმისაწვდომი სხვა გვერდებისთვის. External CSS: სტილები დაწერილია ცალკე .css ფაილში და მიერთებულია HTML გვერდზე ტეგის გამოყენებით. მაგალითი: styles.css ფაილში: p { color: green; font-size: 16px; } დადებითი: ყველაზე ეფექტური მეთოდი სტილების მართვისთვის მრავალ გვერდზე. ნაკლოვანება: საჭიროებს დამატებით ფაილებს და სერვერზე მათ ჩატვირთვას. 2) CSS-ის ძირითადი ტერმინები და სელექტორები: დეკლარაცია (Declaration): აღწერს, როგორ უნდა გამოიყურებოდეს ელემენტი. შედგება თავსართის (property) და მნიშვნელობის (value) წყვილისგან. მაგალითი: color: red; font-size: 20px; ამ დეკლარაციის მიხედვით, ტექსტი იქნება წითელი და ზომა - 20px. სელექტორი (Selector): განსაზღვრავს, რომელ ელემენტზე უნდა მოქმედებდეს CSS-ის სტილები. მაგალითი: p { color: blue; } p არის სელექტორი, რომელიც არჩევს ყველა

ელემენტს. სელექტორების ტიპები და პრიორიტეტულობა: ID სელექტორი (#): მიმართავს ელემენტს უნიკალური id-ით. მაგალითი: #header { background-color: yellow; }

ჩემი სათაური
პრიორიტეტი: უფრო მაღალი, ვიდრე კლასისა და ელემენტის სელექტორებს. Class სელექტორი (.): მიმართავს ელემენტს class ატრიბუტის მიხედვით. შეიძლება გამოყენებულ იქნას მრავალ ელემენტზე. მაგალითი: .button { color: white; background-color: black; } Element სელექტორი: მიმართავს ყველა ელემენტს მისი ტეგის სახელის მიხედვით. მაგალითი: p { font-size: 18px; }

ტექსტი

პრიორიტეტულობა (Specificity): ID (#): უმაღლესი პრიორიტეტი. Class (.) ან ატრიბუტები: საშუალო პრიორიტეტი. Element: ყველაზე დაბალი პრიორიტეტი. თუ ელემენტს აქვს კონფლიქტური სტილები, პრიორიტეტი ენიჭება ID-ს, შემდეგ კლასს და ბოლოს ელემენტს. მაგალითი: p { color: blue; } #special { color: red; }

ტექსტი

ეს ტექსტი იქნება წითელი, რადგან ID სელექტორს აქვს უმაღლესი პრიორიტეტი.
